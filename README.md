# snsu-tracker

## Problem:

Improve handoff communication between teams over the weekend on 24/5 and 24/7 lines. Sign off sheet for tasks

## Tool Description:

An MES tool/board to track Weekend Ops with tasks, progress, and delay communication.

## Required Functionality:

- All Line Status Board
- Time to complete in backend data/export
- Expected time to run displayed as deadline for execution team (from MES)
- Sign off sheet for task completion
- FUTURE to list subtasks for even more granular process tracking

## Project Requirements

- EOM March
- On MES Board Q2
- accessible by all departments
- UI separate by shift hours and Spirits
- Usable by mobile (IPhone 12+, MS Surface)
- OPTIONAL Automated Notification (MS Teams)
- TV Screen View

## Failures to mitigate:

- not seeing board
- pencil whipping
- over-use of notifications
